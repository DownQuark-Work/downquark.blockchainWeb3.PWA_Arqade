<h1>Test to prevent UI bugs</h1>
<p>As a dependency of many apps, design systems are a single point of failure.<br/>
Introducing UI bugs in common components snowballs into major regressions across projects.<br/>
Prevent regressions by automating visual and unit tests to run whenever you push code.</p>
