<h1>Review to reach consensus and gather feedback</h1>
<p>Design systems effect every product and feature in an organization. As such, they require alignment between developers, designers, product managers, and other disciplines.</p>
<p>The most effective way to reach alignment is to publish Storybook online regularly as a part of your CI process. Allowing all stakeholders have a universal reference point for UI implementation - and no one touches code or the command line.</p>
<p>There's even a way to track consenus within Storybook.</p>
