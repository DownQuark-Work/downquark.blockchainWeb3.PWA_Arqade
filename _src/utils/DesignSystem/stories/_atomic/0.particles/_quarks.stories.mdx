DownQuark&rsquo;s Abstracted Integrations.

## Quarks
> - Quark values are abstractions of all other tokens
>   - Choosing a Quark sets the "look and feel" of that section of the d?app
>   - Related Quark values are themeable

## Usage
> The main goal of having everything abstracted to this level is to allow one more at page level.
> To set the defaults on the page you simply remove the defined quark.
> e.g.
```
:root {
--dq-accent-primary: var(--dq-quark-hub-accent-primary);
--dq-accent-secondary: var(--dq-quark-hub-accent-secondary);
--dq-accent-tertiary: var(--dq-quark-hub-accent-tertiary);
--dq-color-darker: var(--dq-quark-hub-color-darker);
--dq-color-dark: var(--dq-quark-hub-color-dark);
--dq-color-default: var(--dq-quark-hub-color-default);
--dq-color-light: var(--dq-quark-hub-color-light);
--dq-color-lighter: var(--dq-quark-hub-color-lighter);
--dq-color-complement: var(--dq-quark-hub-color-complement);
--dq-typography-h1-font: var(--dq-quark-hub-typography-h1-font);
--dq-typography-h1-margin: var(--dq-quark-hub-typography-h1-margin);
  ...
--dq-typography-text-font: var(--dq-quark-hub-typography-text-font);
--dq-typography-text-margin: var(--dq-quark-hub-typography-text-margin);
}
```
For each page/section/quark  you should _only_ reference the top level abstractions. Without using any quark definition (`quark-hub`)
