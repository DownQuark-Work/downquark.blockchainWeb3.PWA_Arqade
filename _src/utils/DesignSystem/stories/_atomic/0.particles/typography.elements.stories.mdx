## Usage

<Usage>
  <Correct>
    Use semantic tags in the correct order ie: h1 â†’ h2. For style overrides,
    reference a tag as a utilitity ie: class="header4"
  </Correct>
  <Incorrect>Skip semantic tag levels for visual style ie: h1 â†’ h4</Incorrect>
</Usage>
<Usage>
  <Correct>Rely on the base theme for Neucha which is set on the body</Correct>
  <Incorrect>
    Reference font-family in your CSS, it's usually unnecessary
  </Incorrect>
</Usage>
<Usage>
  <Correct>Rely on the base theme for link styles</Correct>
  <Incorrect>
    Override link styles change color other than for
    navigational components
  </Incorrect>
</Usage>
<Usage>
  <Correct>
    Use semantic tags for font-size first, then utility classes, then custom CSS
    in the form of rems or ems
  </Correct>
  <Incorrect>
    Set arbitrary font-sizes without using semantic tags, and especially don't
    use px values
  </Incorrect>
</Usage>
